Гастрономический Рай — статический сайт (No-Build)
====================================================

Как задеплоить на Netlify (быстро, через Drop):
1. Зайдите на https://app.netlify.com/drop
2. Перетащите сюда содержимое папки `gastro-paradise-site` (или zip-архив `gastro-paradise-site.zip`).
3. Netlify выдаст URL вашего сайта.

Файлы:
- index.html — весь сайт (React через CDN, hash-роутинг)
- assets/logo.png — логотип (можете заменить своим файлом с тем же именем)

Изменения:
- Редактируйте данные события прямо в index.html внутри объектов `event` и `e`.
